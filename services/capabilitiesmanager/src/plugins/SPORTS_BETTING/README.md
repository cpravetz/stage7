# SPORTS_BETTING Plugin

Sports betting analysis and prediction plugin with odds calculation and risk assessment.

## Features

- Odds analysis and conversion (American/Decimal)
- Probability calculation
- Expected value analysis
- Risk assessment and bankroll management
- Betting history tracking
- Game prediction generation

## Supported Sports

- football
- basketball
- baseball
- soccer
- hockey
- tennis

## Bet Types

- moneyline
- spread
- over_under
- parlay
- teaser

## Actions

- `analyze_odds`: Analyze sports odds
- `calculate_probability`: Calculate win probabilities
- `assess_risk`: Assess bet risk profile
- `track_bets`: Track betting history and performance
- `generate_predictions`: Generate game predictions
