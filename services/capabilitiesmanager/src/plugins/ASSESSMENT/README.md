# ASSESSMENT Plugin

Assessment and testing plugin for creating, grading, and analyzing tests and assessments.

## Features

- Assessment creation with multiple question types
- Automated grading with multiple scales
- Comprehensive result analysis
- Knowledge gap identification
- Remediation plan generation
- Detailed assessment reports

## Supported Question Types

- multiple_choice
- short_answer
- essay
- true_false
- matching
- fill_blank

## Grading Scales

- percentage (0-100)
- letter (A-F)
- GPA equivalent (0-4.0)

## Actions

- `create_assessment`: Create new assessment
- `grade_assessment`: Grade completed assessment
- `analyze_results`: Analyze assessment results
- `identify_gaps`: Identify knowledge gaps
- `suggest_remediation`: Generate remediation plan
- `generate_report`: Generate comprehensive report
