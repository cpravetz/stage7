# SCRIPTWRITER_CONTENT Plugin

Script writing assistance plugin for screenwriting and dialogue generation.

## Features

- Script generation and outline creation
- Scene structure analysis
- Dialogue suggestion based on context
- Screenplay formatting validation
- Pacing analysis
- Comprehensive script reporting

## Supported Script Types

- feature (up to 180 pages)
- tv_episode (up to 60 pages)
- short (up to 30 pages)
- web_series (up to 15 pages)

## Actions

- `generate_script`: Generate script outline
- `analyze_structure`: Analyze script structure
- `suggest_dialogue`: Get dialogue suggestions
- `check_formatting`: Validate formatting
- `analyze_pacing`: Analyze script pacing
- `generate_report`: Generate comprehensive script report
