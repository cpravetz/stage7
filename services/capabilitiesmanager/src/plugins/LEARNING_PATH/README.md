# LEARNING_PATH Plugin

Personalized learning path creation and management plugin.

## Features

- Personalized learning path generation
- Module management and progression
- Progress tracking
- Mastery assessment
- Learning gap identification
- Next module recommendations
- Comprehensive learning reports

## Supported Subjects

- python
- javascript

## Learning Levels

- beginner
- intermediate
- advanced
- expert

## Actions

- `create_path`: Create personalized learning path
- `add_module`: Add module to path
- `track_progress`: Track learning progress
- `assess_mastery`: Assess mastery level
- `recommend_next`: Get next module recommendation
- `generate_report`: Generate learning report
