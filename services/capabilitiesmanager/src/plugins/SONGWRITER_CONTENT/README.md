# SONGWRITER_CONTENT Plugin

Song composition assistance plugin for creating lyrics and analyzing song structure.

## Features

- Lyric generation for multiple genres
- Melody suggestions
- Song structure analysis
- Rhyme scheme checking and analysis
- Improvement suggestions
- Comprehensive song report generation

## Supported Genres

- pop
- hip_hop
- ballad
- folk

## Actions

- `generate_lyrics`: Generate song lyrics
- `suggest_melody`: Get melody recommendations
- `analyze_structure`: Analyze song structure
- `check_rhyme_scheme`: Check rhyme scheme quality
- `suggest_improvements`: Get improvement suggestions
- `generate_report`: Generate comprehensive song report
